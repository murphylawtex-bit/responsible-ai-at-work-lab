# Responsible AI at Work Lab

Responsible AI at Work Lab builds practical open tools for small and mid-sized organizations adopting AI in real business workflows.

The first project is the **Responsible AI Adoption Toolkit for SMEs**: a set of templates, guides, examples, and risk-management materials that help organizations measure productivity while protecting human judgment, job quality, privacy, transparency, and public trust.

## Core toolkit

- AI use-case intake
- Stakeholder and impact map
- Productivity / ROI measurement plan
- Workforce impact checklist
- Risk register
- Human oversight plan
- Lightweight responsible AI governance policy
- Pilot review scorecard
- NIST AI RMF crosswalk
- 30-day implementation guide
